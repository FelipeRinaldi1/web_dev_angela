# Box Model

cada elemento de uma pagina é uma caixa.

border: px,styele,color;
border-top:0px;
border-width: (top) (right) (bottom) (left)
caso seja passado 2 valores apenas, o comando dara a seguinte resposta: 1 + 3 , 2 + 4 (top/botton) (left/right)

padding:20px;
empurra as bordas de uma caixa em 20 pixels simetricamente (margem interna, preenchimento)

margin:10px;
cria uma margem de 10 pixels em volta da caixa do elemento

padding - margin - border - width
